INSTRUCCIONES

Bienvenido a Lunar Lander, tu misión consiste en aterrizar el cohete a la velocidad adecuada para que no explote, tienes combustible limitado y cuentas con un medidor de velocidad y altura para guiarte, pulsa cualquier tecla para encender el motor y deja de pulsar para apagarlo o clica en la pantalla para cambiar de apagado a encendido y viceversa.

Buena suerte! (Click aquí para continuar)

DIFICULTAD

  • FACIL: Aterriza a 5m/s o menos con 100L de combustible.
  • NORMAL: Aterriza a 2'5m/s o menos con 50L de combustible.
  • CHUNGO: Aterriza a 1m/s o menos con 50L de combustible.
